# M & G Traders Dashboard

A modern, responsive dashboard for managing company funds, bills, multi-user notices, and employee tracking.

## Features
- **Dynamic Database**: Powered by SQLite for real-time data persistence.
- **Role-Based Access**: Specialized views for Admins, Superadmins, and Employees.
- **Fund Automation**: Real-time balance updates based on bills and payments.
- **Mobile Friendly**: Fully responsive design with an adaptive sidebar.
- **Notice Board**: Shared notices for the entire team.
- **Employee Tracking**: Track employee status and current duties.

## Tech Stack
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
- **Backend**: Node.js, Express
- **Database**: SQLite3

## Local Setup
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the server with `npm start` or `npm run dev`.
4. Access the dashboard at `http://localhost:3000`.

## Deployment
Configured for **Vercel** deployment with `vercel.json` included.
